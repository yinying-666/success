# 个人结课作业：在线记事本Web系统
提交人：尹颖
班级：软件24-2

## 项目整体功能
核心功能：网页实现**添加笔记、查看全部笔记**
1. 前端表单填写标题、内容提交新增笔记
2. 页面实时展示所有已保存笔记
3. 预留学校API密钥配置，可在校园网对接校内接口
4. 项目可部署到Railway生成公网访问链接

## 项目文件说明
- main.py：原有基础问候函数 + 记事本增查核心逻辑
- app.py：Flask网页后端入口，处理页面请求
- templates/index.html：记事本前端网页页面
- .env：存放学校API配置（测试密钥占位，等待老师下发正式密钥）
- requirements.txt：Python运行依赖清单
- Procfile：Railway部署启动配置文件
